Инструкция по сборке проекта:
Для сборки проекта требуется установленные QT 5.0.0 +  и CMake 3.18+

1) Откройте файл CMakeList.txt и в строках 
set (QT_DIR D:/Work/C++/6.2.2/msvc2019_64)
set(CMAKE_PREFIX_PATH "D:/Work/C++/6.2.2/msvc2019_64")
замените пути к директориям на собственные и сохраните файл. 
2)Используя CMake, соберите проект в удобную для вас директорию
